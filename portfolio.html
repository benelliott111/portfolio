Travel Application

Live Application

Repository

This application allows the user to search any city on the planet and returns weather, currency exchange rates in CAD, popular destinations in the area and a working map of the region. The homepage contains a search bar, as well as three popular cities that the user can navigate to with a single click. Upon searching, the user will be given up to twelve results for the city. If there are fewer than twelve matches, more recommended cities will be pulled from a hardcoded database. The user may then choose the city they would like to visit and will be shown the full suite of information as described above. Favourite cities can be saved in local storage and accessed in the navbar at any time. The application uses Bootstrap, CSS, HTML and Javascript and was built with 4 other programmers.

Quiz Application

Live Application

Repository

This application runs a quiz to test the users hockey knowledge. The quiz begins when the user clicks the start button, and will continue until the score reaches 0. Answers are given in multiple choice format, with wrong answers reducing the score by ten. After answering all questions the user my enter their initials and log their score. The user score and initial will be saved and the user can try the quiz again at their leisure. The application uses Bootsrap, CSS, HTML and Javascript.

Weather API Application

Live Application

Repository

This application uses the openweathermap.org API to fetch information on the city that the user searches. Upon searching, the user will be shown the weather icon, temperature, humidity, wind speed and UV index as well as the five day forecast for the city. Each city the user searches is added into local storage and displayed beneath the search bar to allow for easy navigation to past searches. The application uses Bootstrap, HTML, CSS and Javascript.
